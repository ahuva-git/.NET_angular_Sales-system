## 🚀 How to Run the Project

### 1. Prerequisites
Make sure you have the following installed:
* [.NET 8 SDK](https://dotnet.microsoft.com/download/dotnet/8.0)
* [Node.js](https://nodejs.org/)
* [Angular CLI](https://angular.io/cli)

### 2. Setting up the Backend (Server)
1. Open a terminal (cmd) in the `WEBAPI` folder.
2. Run the following command to start the server:
   ```bash
   dotnet run
3. The API will be running at: http://localhost:5000 (check your console for the exact port).
4. You can view the API documentation at: http://localhost:5000/swagger.

### 3. Setting up the Frontend (Client)
1. Open a terminal (cmd) in the Angular folder.
2. Install dependencies (run this only the first time):
   ```bash
    npm install
3. Run the application:
   ```bash
    ng serve
4. Open your browser and go to: http://localhost:4200.
